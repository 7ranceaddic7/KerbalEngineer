Description

This plugin will allow you to view important statistics about your ship on the fly, whilst building it and in flight.

Installation

  • Copy the 'KerbalEngineer' folder into the 'GameData' folder located within your Kerbal Space Program installation directory.
  • Attach an Engineer part from the science tab to your ship.
    • ER7500 includes flight engineer.
